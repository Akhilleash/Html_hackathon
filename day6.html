üé• Video tag (<Video>)

The <video> tag is used to embed video files.

Modern browsers support MP4, WebM, and Ogg formats.

You can provide multiple elements ‚Üí the browser picks the one it support

  • controls ‚Üí Adds UI (play, pause, volume, fullscreen).

  • autoplay ‚Üí Plays video automatically when the page loads.

  • loop ‚Üí Restarts video when it ends.

  • muted ‚Üí Starts video with no sound.

  • poster ‚Üí Shows a thumbnail before the video starts.

  • width / height ‚Üí Size of the video player.

Example

You can use the video tag using

<video src="my-awesome-video.mp4"> </video>

Header,Main,Footer

Header

The top section of a webpage or a section.

Usually contains:

Website logo or title

Navigation menu

Taglines or introductory text

Main

The main content area of a webpage.

Only one <main> is allowed per page.

It should not include things like navigation, sidebars, or repeated content

(those go in <header>, <footer>, or <aside>).

Footer

The bottom section of a webpage or section.

Usually contains:

Copyright info

Contact info

Social media links

Related site links Like <header>, you can have multiple footers (one for the page and one for a section).

Deploying First Website

Step by Step: Deploy Website on Netlify

  • Step 1: Prepare your website files

    Make sure all HTML, CSS, JS, images, and other assets are in one folder. The main page should be named index.html.
  • Step 2: Sign in to Netlify

    Go to Netlify and log in using GitHub, GitLab, Bitbucket, or email.
  • Step 3: Deploy using Drag & Drop

    Click "Add new site" ‚Üí "Deploy manually" ‚Üí "Drag & Drop". Drag the folder containing your website files into the drop area. Netlify will generate a temporary URL.
  • Step 4: Test your site

    Open the generated Netlify URL and check that all pages, images, and scripts load correctly.
  • Step 5: Add a custom domain (optional)

    Go to Domain settings ‚Üí Add custom domain. Update your DNS records to point to Netlify or use Netlify DNS. HTTPS will be provided automatically.

  • Step 6: Update or redeploy

    If you make changes, update your files and drag & drop again. Netlify overwrites the previous deployment.
